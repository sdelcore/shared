Getting started

Put an index.html in a folder, then ship it:

# deploy a directory as a site
shared deploy ./mysite --name mysite

# it is live immediately at
http://mysite.localhost:8787/

Every site can include <script src="/shared.js"></script> to get the shared API for free: a document database with realtime subscriptions, an AI chat proxy, file uploads, websocket channels, and identity.
